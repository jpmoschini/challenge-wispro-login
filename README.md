# Challenge Wispro

### 1- In the project directory, go to:

### `cd client`

### `npm i`

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### 2- Then the project directory, go to:

### `cd server`

### `npm i`

### `node server.js`
